## Image Guidelines

### Technical Requirements
- **Format:** WebP preferred, JPG acceptable
- **Optimization:** Compress all images before uploading
- **Naming:** Use kebab-case, descriptive filenames
- **Alt Text:** Always provide meaningful alt text in MDX frontmatter

### Dimensions
- **Featured Images:** 1200x675px (16:9 ratio)
- **Open Graph Images:** 1200x630px
- **Author Photos:** 400x400px (square)
- **Logo:** Maintain aspect ratio, provide multiple sizes

### Content Guidelines
- Use high-quality, professional photography
- Avoid stock photos that look generic
- Show authentic military/athletic training settings when possible
- Ensure all images have appropriate usage rights

## Adding New Images

1. Optimize image file size (target < 200KB for featured images)
2. Add to appropriate subdirectory
3. Reference in MDX frontmatter with full path from `/images/`
4. Include descriptive alt text for accessibility
